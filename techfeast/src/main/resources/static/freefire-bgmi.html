Gaming Arena — FreeFire / BGMI

About the Event

Gaming Arena brings competitive esports action to the campus. Participants will compete in intense battle royale matches where strategy, teamwork, reflexes, and survival skills determine the winner. This is not casual gaming — this is competitive survival.

Match Rules

  • Official tournament room will be created by organizers.
  • All participants must join on time.
  • Use of hacks, scripts, or unfair tools leads to immediate disqualification.
  • Emulator usage only if permitted in the official rulebook.
  • Players are responsible for maintaining a stable internet connection.

Game Titles

  • BGMI (Battlegrounds Mobile India)
  • Free Fire

Team Specifications

  • Squad Mode Only — 4 Players per Team
  • Solo Mode Removed
  • All players must carry valid college ID

Event Format

  • League / Qualification Round
  • Semi-Finals
  • Grand Finals

Maps and match settings will be announced before the match.
